# 🔗 URL Shortify

A clean and simple **URL Shortener** built with **Node.js**, **Express.js**, **MongoDB**, and **ShortID**.  
It allows users to convert long URLs into short, shareable links with a fast and responsive interface.

---

## ✨ Features

- Shorten long URLs instantly
- Generate unique short links using **ShortID**
- Store original and shortened URLs in **MongoDB**
- Redirect users to the original URL
- Clean and responsive frontend
- Easy environment variable setup

---

## 🛠 Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**
- **ShortID**
- **HTML**
- **CSS**
- **JavaScript**
